## Rental System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/29351a6b6d604747822f244f6364ecc9/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Property Information
5. Property Viewing Appointment
6. Favorite Property Information
7. Comment on Property Information
8. Forum Posting and Commenting
9. Property News
10. Personal Center
11. My Posts
12. My Favorites
13. Online Customer Service
14. My Property Viewing Appointments
15. Online Contract Signing
16. Rental Reviews



**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Property Type Management
5. Property Information Management
6. Property Viewing Appointment Management
7. Online Contract Signing Management
8. Rental Review Management
9. Forum
10. Property News Management
11. Online Customer Service
12. Carousel Management
